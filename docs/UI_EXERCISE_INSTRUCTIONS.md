# UI Automation Test Exercise

**Objective:** Automate website interactions on Airalo's website and verify eSIM package pricing.

## Test Case

### 1. Open Airalo's Website

- Launch a browser and navigate to Airalo's website.

### 2. Search for Japan

- In the search field on the home page, type "Japan" and click on the result with the Japanese flag.

### 3. Select an Unlimited eSIM Package

- On the next page, click on the unlimited data plan tab.
- Select the 7 days unlimited.

### 4. Verify Package Price

- Verify that the price of the 7-day unlimited data package matches the price next to the Buy now button.
